# CMIP6 Climate Analysis with Python

## 1. Project Overview (项目概述)
This project is part of my undergraduate thesis at the University of Reading/NUIST. It focuses on analyzing and visualizing **CMIP6 ocean biogeochemistry data** to understand spatial patterns and climate trends.

## 2. Key Features (核心功能)
- **Data Processing**: Automated workflows for handling large-scale NetCDF4 datasets.
- **Statistical Analysis**: Calculation of zonal means, seasonal cycles, and regional anomalies.
- **Visualization**: High-quality geospatial plotting using `Cartopy` and `Matplotlib`.

## 3. Tech Stack (技术栈)
- **Language**: Python 3.x
- **Core Libraries**: `xarray`, `numpy`, `pandas`, `cartopy`, `matplotlib`.
